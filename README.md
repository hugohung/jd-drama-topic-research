# 🔥 热门话题及短剧题材调研专家

> WorkBuddy Skill — 自动搜集18个平台热榜，分析短剧适配度，生成题材推荐报告

## 功能特性

- **多平台热点抓取**：覆盖微博、知乎、小红书、B站、抖音等18个平台实时热榜
- **智能题材映射**：从热点中筛选适合改编短剧的话题，按情绪强度/共鸣广度/冲突密度评分
- **红果热播对标**：结合红果短剧热播榜分析同类题材市场表现
- **风险管控**：提示政策风险、题材同质化风险
- **创作辅助**：生成分集大纲、人物小传、剧本框架
- **京东植入建议**：支持自然融入/硬广/彩蛋等多种植入方式
- **环境自检**：自动检测 MCP 工具和 API Key，缺失时主动引导配置

## 安装方式

### WorkBuddy 用户

1. 下载 [Release zip](../../releases/latest)
2. 在 WorkBuddy 技能管理 → 上传技能，选择 zip 文件

### 从源码安装

```bash
git clone https://github.com/hugohung/workbuddy-skill-drama-topic-research.git ~/.workbuddy/skills/drama-topic-research
```

## 使用方式

在 WorkBuddy 对话中直接说：

> "帮我调研一下最近的短剧题材"
>
> "我想做一个关于职场逆袭的短剧"
