## 📊 核心能力

本 Skill 是一个专业的短剧题材分析师和AI创作辅助专家，核心能力包括：

### 1️⃣ 热点→题材映射
不只搜集热榜，更要分析哪些话题**适合改编短剧**

### 2️⃣ 智能适配分析
评估话题的短剧适配度：
- ⭐ **情绪强度** — 是否能引发强烈情绪（愤怒/感动/爽）
- ⭐ **共鸣广度** — 是否能引发大量用户共鸣
- ⭐ **冲突密度** — 是否包含足够的戏剧冲突
- ⭐ **改编空间** — 是否能扩展成多集剧情

### 3️⃣ 红果热播对标
结合**红果短剧热播榜**，分析同类题材的市场表现

### 4️⃣ 风险管控
提示**政策风险**、**题材同质化风险**

### 5️⃣ 创作辅助
生成**分集大纲**、**人物小传**、**剧本框架**

## 🎯 适用场景

- 📝 **题材策划**：寻找下一个爆款短剧方向
- 🔍 **热点追踪**：实时掌握社交媒体热门话题
- 📊 **市场分析**：了解红果平台同类题材表现
- ✍️ **剧本前期**：快速生成题材推荐报告和创作框架

## 🚀 快速开始

### 前置要求

#### 1. MCP 工具（推荐）
安装 [`mcp-hotnews-plus`](https://www.codebuddy.cn/mcp/mcp-hotnews-plus) 可更高效地获取多平台热榜数据

#### 2. 52api.cn API Key（可选）
获取红果短剧热播榜数据需要此 Key：
- 访问 https://www.52api.cn 注册账号
- 进入控制台 → 密钥管理 → 创建密钥
- 将 Key 告诉我，我会自动保存配置

> 💡 没有 API Key 也能使用，红果对标分析会自动降级为通用模板分析

### 使用示例

在 WorkBuddy 对话中直接说：

```
帮我调研一下最近的短剧题材
```

```
我想做一个关于职场逆袭的短剧
```

## ⚠️ 注意事项

1. **AI 生成内容需审核**：本 Skill 生成的题材推荐、分集大纲等仅供参考，正式拍摄前务必进行人工审核和二次创作
2. **政策风险**：避免敏感话题（政治、宗教、色情、暴力），注意价值观导向
3. **同质化风险**：热门题材容易被扎堆制作，建议加入差异化设定
4. **数据时效性**：热榜数据实时更新，建议每天重新调研
